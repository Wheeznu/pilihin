Developer 4 — Admin Content Management

Sebagai Developer 4 dalam pembangunan website Pilih.in, saya bertanggung jawab atas pengelolaan konten platform. Berikut adalah 10 Interface yang telah saya kerjakan.

1. Profil Admin

Halaman profil administrator yang menampilkan informasi pribadi admin seperti nama, email, ID Admin, tanggal bergabung, dan aktivitas terakhir. Dilengkapi dengan avatar beranimasi (ring berputar), form edit profil untuk mengubah data diri, serta statistik ringkasan total film, pengguna, dan ulasan.

  • Fitur: Edit profil, upload foto, statistik, aktivitas terbaru
  • Status: ✅ Selesai

2. Kelola Konten

Halaman daftar semua film dalam bentuk tabel yang dilengkapi fitur pencarian, filter status (published/draft), pagination, serta tombol Edit dan Hapus. Admin dapat melihat poster, judul, rating, release date, jumlah pemeran, dan status film.

  • Fitur: Tabel film, search, filter status, pagination, edit, hapus
  • Status: ✅ Selesai

3. Tambah Artikel

Formulir untuk menambahkan artikel/berita baru ke platform. Berisi field: Judul Artikel, Kategori (Berita/Artikel), Thumbnail (upload gambar), Konten (textarea), Penulis, dan Tag. Dilengkapi preview konten dan tombol simpan.

  • Fitur: Upload thumbnail, pilih kategori, preview konten
  • Status: ✅ Selesai

4. Edit Artikel / Berita

Halaman untuk mengedit artikel yang sudah ada. Data diambil berdasarkan ID artikel, kemudian form diisi otomatis. Admin dapat mengubah judul, kategori, thumbnail, konten, penulis, dan tag. Tersedia tombol Simpan Perubahan dan Hapus Artikel dengan konfirmasi modal.

  • Fitur: Load data otomatis, edit form, hapus dengan konfirmasi
  • Status: ✅ Selesai

5. Tambah Film

Formulir lengkap untuk menambahkan film baru ke database. Berisi field: Judul Film, Sinopsis, Release Date, Rating (bintang 1-5), Link Trailer, Link Video Film, dan Upload Poster (dengan preview). Juga terdapat fitur tambah aktor dinamis (tag) dan progress bar kelengkapan form.

  • Fitur: Upload poster, tambah aktor dinamis, progress bar, validasi form
  • Status: ✅ Selesai

6. Edit Film

Halaman untuk mengedit data film yang sudah ada. Data diambil dari URL parameter ?id=xxx, kemudian form diisi otomatis. Admin dapat mengubah judul, sinopsis, rating, release date, link trailer, link video, aktor, dan poster. Tersedia tombol Simpan Perubahan dan Hapus Film dengan konfirmasi modal.

  • Fitur: Load data otomatis, edit form, hapus dengan konfirmasi
  • Status: ✅ Selesai

7. Edit Aktor

Halaman daftar semua aktor dalam bentuk tabel. Menampilkan foto, nama, tanggal lahir, jumlah film yang dibintangi, dan status. Admin dapat mencari aktor, mengedit data aktor, atau menghapus aktor dengan konfirmasi modal. Terdapat pagination untuk navigasi data.

  • Fitur: Search, pagination, edit, hapus dengan konfirmasi
  • Status: ✅ Selesai

8. Tambah Aktor

Formulir untuk menambahkan aktor baru ke database. Berisi field: Nama Aktor, Bio, Tanggal Lahir, Upload Foto Aktor, dan Film yang Dibintangi (dropdown dari database film yang sudah ada). Layout dua kolom dengan tips pengisian.

  • Fitur: Upload foto, pilih film dari dropdown, simpan ke localStorage
  • Status: ✅ Selesai

9. Edit Sutradara

Halaman daftar semua sutradara dalam bentuk tabel. Menampilkan foto, nama, tanggal lahir, jumlah film yang disutradarai, dan status. Admin dapat mencari, mengedit, atau menghapus sutradara. Fitur edit akan membawa ID ke halaman edit, sedangkan hapus akan memunculkan modal konfirmasi.

  • Fitur: Search, pagination, edit, hapus dengan konfirmasi
  • Status: ✅ Selesai

10. Tambah Sutradara

Formulir untuk menambahkan sutradara baru. Berisi field: Nama Sutradara, Bio, Tanggal Lahir, Upload Foto Sutradara, dan Film yang Disutradarai (dropdown dari database film yang sudah ada). Layout dua kolom dengan tips pengisian di sisi kanan.

  • Fitur: Upload foto, pilih film dari dropdown, layout estetik
  • Status: ✅ Selesai

Kesimpulan

Sebagai Developer 4, saya telah menyelesaikan 10 interface utama untuk pengelolaan konten platform Pilih.in. Mulai dari Profil Admin, Kelola Konten, pengelolaan artikel, pengelolaan film (tambah & edit), pengelolaan aktor (tambah & edit), hingga pengelolaan sutradara (tambah & edit). Seluruh halaman dirancang dengan tampilan estetik, responsif, dan mendukung fungsionalitas CRUD lengkap. Pengerjaan ini memastikan bahwa admin dapat mengelola seluruh konten platform dengan mudah dan efisien.
