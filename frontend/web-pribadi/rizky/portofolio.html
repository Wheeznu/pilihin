01 Stack (Tumpukan) & Delimiter ▾

LIFO Push / Pop Stack (Tumpukan) adalah struktur data linear yang mengikuti prinsip Last In, First Out (LIFO). Analogi sederhananya adalah tumpukan piring; piring terakhir yang diletakkan di atas tumpukan akan menjadi piring pertama yang diambil. Penambahan elemen baru (push) dan penghapusan elemen (pop) hanya bisa dilakukan pada satu ujung yang sama, yaitu bagian puncak (Top).

Dalam ilmu komputer, konsep ini adalah fondasi bagi manajemen memori (Call Stack), fitur Undo pada aplikasi, serta fungsi *parsing* sintaks. Salah satu penerapannya adalah algoritma pengecekan kurung (delimiter) untuk mengevaluasi apakah sebuah ekspresi matematika ditulis dengan urutan buka-tutup kurung yang seimbang dan valid.

Operasi Dasar Stack Prosedural

Python 
# Operasi Dasar Stack Prosedural
def buat_stack(): return []
def push(stack, item): stack.append(item)
def pop(stack): return stack.pop()
def is_empty(stack): return stack == []
def peek(stack): return stack[-1] if not is_empty(stack) else None

Studi Kasus: Pengecekan Kurung (Parentheses)

Algoritma untuk memastikan jumlah kurung buka dan tutup dalam operasi matematika seimbang dan valid.

Python 
def checkParentheses(strMat):
    data = buat_stack()
    for ch in strMat:
        if ch == '(':
            push(data, ch)
        elif ch == ')':
            if is_empty(data):
                return "Kelebihan kurung tutup"
            pop(data)

    if is_empty(data):
        return True
    else:
        return "Kelebihan kurung buka"
02 Queue (Antrean) & Sparse Matrix ▾

FIFO Enqueue / Dequeue Queue (Antrean) beroperasi dengan prinsip First In, First Out (FIFO). Elemen yang masuk lebih dulu akan diproses dan dikeluarkan lebih dulu. Penambahan data (enqueue) terjadi di sisi belakang (Rear), sedangkan penghapusan data (dequeue) terjadi di sisi depan (Front). Pola ini sangat krusial dalam sistem task scheduling CPU, antrean mesin cetak (printer), hingga pemrosesan pesan (message brokering).

Selain Queue, Sparse Matrix adalah teknik struktur data tingkat lanjut untuk efisiensi memori. Dalam matriks 2D berukuran raksasa yang mayoritas elemennya bernilai nol, menyimpan seluruh elemen tersebut akan memboroskan kapasitas. Dengan konsep Sparse Matrix, kita memanfaatkan Dictionary untuk memetakan koordinat baris dan kolom hanya pada elemen yang memiliki nilai aktual, membuang alokasi memori untuk angka nol.

Operasi Dasar Queue Prosedural

Python 
# Operasi Dasar Queue Prosedural
def queue(): return []
def enqueue(q, data): q.append(data)
def dequeue(q): return q.pop(0) if q else None

03 Eksplorasi Fungsi Rekursif ▾

Base Case Call Stack Rekursi adalah teknik pemrograman di mana sebuah fungsi memanggil dirinya sendiri secara berulang untuk memecah masalah besar menjadi sub-masalah yang lebih kecil namun identik. Komponen paling vital dari rekursi adalah Base Case (kondisi dasar); sebuah batas henti agar fungsi tidak memanggil dirinya terus-menerus yang dapat menyebabkan kelebihan beban memori (Stack Overflow).

Pendekatan ini menghasilkan kode yang sangat deklaratif dan elegan, terutama untuk persoalan matematis. Misalnya, dalam konversi bilangan desimal ke biner, fungsi rekursif dapat menggunakan parameter num secara dinamis, membagi nilai parameter tersebut menjadi separuhnya pada setiap iterasi hingga mencapai kondisi dasar, sebelum akhirnya mencetak jejak hasil modulus.

1. Faktorial Rekursif

Python 
def factorialRekursif(bilangan):
    if bilangan <= 1: return 1  # Base Case
    else: return bilangan * factorialRekursif(bilangan - 1)

2. Eksponen (Pangkat) Rekursif

Python 
def eksponenRekursif(x, n):
    if n == 0: return 1  # Base case pangkat 0
    else: return x * eksponenRekursif(x, n - 1)

3. Konversi Desimal ke Biner

Python 
def konversiBiner(num):
    if num > 1:
        konversiBiner(num // 2)
    print(num % 2, end='')
04 Sorting & Modifikasi Lanjut ▾

Bubble Sort Selection Sort Sorting (Pengurutan) adalah proses algoritmik mengatur sekumpulan data ke dalam urutan tertentu (Ascending atau Descending). Berbagai metode digunakan tergantung kompleksitas masalahnya:

  • Bubble Sort: Algoritma klasik yang membandingkan pasangan elemen yang bersebelahan, lalu menukarnya jika posisinya terbalik. Mirip gelembung udara, nilai terbesar akan pelan-pelan "naik" ke ujung array pada setiap iterasi.
  • Selection Sort: Bekerja dengan cara membagi rentang data menjadi dua bagian (terurut dan belum terurut). Algoritma ini mencari elemen paling minimum/maksimum dari sisi yang belum terurut, dan memindahkannya ke posisi absolutnya.

Dengan menggunakan logika kontrol prosedural (fungsi berbasis blok iterasi murni tanpa pembungkus objek), memodifikasi indeks—seperti memisahkan area pertukaran angka genap dan ganjil—dapat dilakukan dengan sangat presisi.

1. Bidirectional Bubble Sort (Cocktail Sort)

Modifikasi performa: Iterasi ganjil bergerak memindai array dari kiri ke kanan, sedangkan iterasi genap memindai dari kanan ke kiri. Dilengkapi mekanisme early stopping jika data sudah terurut.

Python 
def bidirectionalBubbleSort(arr):
    n = len(arr)
    tukar = True
    awal = 0
    akhir = n - 1

    while tukar:
        tukar = False
        # Kiri ke kanan
        for i in range(awal, akhir):
            if arr[i] > arr[i + 1]:
                arr[i], arr[i + 1] = arr[i + 1], arr[i]
                tukar = True
        if not tukar: break
        tukar = False
        akhir -= 1
        # Kanan ke kiri
        for i in range(akhir - 1, awal - 1, -1):
            if arr[i] > arr[i + 1]:
                arr[i], arr[i + 1] = arr[i + 1], arr[i]
                tukar = True
        awal += 1

2. Selection Sort (Genap Ascending, Ganjil Descending)

Python 
def modifiedSelectionSort(arr):
    n = len(arr)
    for i in range(n):
        if arr[i] % 2 == 0:
            # Genap: Cari nilai minimum untuk Ascending
            idx_target = i
            for j in range(i + 1, n):
                if arr[j] % 2 == 0 and arr[j] < arr[idx_target]:
                    idx_target = j
            arr[i], arr[idx_target] = arr[idx_target], arr[i]
        else:
            # Ganjil: Cari nilai maksimum untuk Descending
            idx_target = i
            for j in range(i + 1, n):
                if arr[j] % 2 != 0 and arr[j] > arr[idx_target]:
                    idx_target = j
            arr[i], arr[idx_target] = arr[idx_target], arr[i]
05 Searching & Analisis Jangkauan ▾

Linear Search Binary Search Searching (Pencarian) bertujuan untuk menemukan keberadaan dan lokasi spesifik dari sebuah target elemen di dalam himpunan data.

  • Linear Search: Memeriksa setiap elemen data satu demi satu secara sekuensial. Keuntungannya adalah dapat bekerja pada himpunan data yang masih acak (belum terurut).
  • Binary Search: Algoritma pencarian super cepat yang membagi ruang array menjadi dua bagian pada setiap iterasinya (Divide and Conquer). Syarat mutlak penggunaannya adalah himpunan data harus sudah dalam kondisi terurut.

1. Modifikasi Linear Search (Multi-Match)

Bukan sekadar menemukan kemunculan pertama, algoritma ini menjelajahi seluruh data untuk menangkap keseluruhan indeks, total frekuensi, serta mencatat kemunculan perdana dan terakhir dari target.

Python 
def multiLinearSearch(arr, target):
    indeks_ditemukan = []
    for i in range(len(arr)):
        if arr[i] == target:
            indeks_ditemukan.append(i)

    if len(indeks_ditemukan) > 0:
        print("Ditemukan di index:", indeks_ditemukan)
        print("Total kemunculan:", len(indeks_ditemukan))
        print("Index pertama:", indeks_ditemukan[0], "Terakhir:", indeks_ditemukan[-1])
    else:
        print("Data tidak ditemukan")

2. Closest Value & Range Search

Python 
def cari_terdekat(arr, target):
    terdekat = arr[0]
    selisih_min = abs(target - arr[0])

    for num in arr:
        selisih = abs(target - num)
        if selisih < selisih_min:
            terdekat = num
            selisih_min = selisih
    return terdekat

def cari_range(arr, lower, upper):
    hasil = []
    for num in arr:
        if lower <= num <= upper:
            hasil.append(num)
    return hasil
06 Hashing, Chaining & Anagram ▾

O(1) Lookup Chaining Collision Hashing adalah fondasi dari pemetaan data dengan kecepatan akses konstan, atau O(1). Data didistribusikan ke dalam sel-sel array bernama Tabel Hash, di mana indeks tujuannya dihitung berdasarkan rumus matematis spesifik (seperti fungsi sisa bagi / modulo).

Problem utama dalam hashing adalah Collision (Tabrakan), yang terjadi manakala dua nilai kunci berbeda menghasilkan indeks penyimpanan yang sama. Implementasi di bawah ini menggunakan teknik resolusi Chaining, yaitu menyarangkan senarai (list) di dalam setiap slot tabel untuk mengakomodir beberapa elemen di indeks yang sama. Pendekatan pada struktur dan eksekusinya berorientasi murni secara prosedural, menghindari enkapsulasi class, guna menjamin ketepatan aliran fungsi secara linear.

1. Pembuatan Tabel & Pencarian (searchHash)

Python 
def create_hash_table(size):
    return [[] for _ in range(size)]

def remainder_function(data, size):
    return data % size

def chaining(arr_data, hash_table):
    for item in arr_data:
        slot = remainder_function(item, len(hash_table))
        hash_table[slot].append(item)

def searchHash(target, hash_table):
    slot = remainder_function(target, len(hash_table))
    for i in range(len(hash_table[slot])):
        if hash_table[slot][i] == target:
            return f"Data di slot ke-{slot}, indeks ke-{i}"
    return False

2. Deteksi Anagram (Implementasi Dictionary/Hash)

Pengembangan pemetaan Hash yang menguji apakah dua untai teks merupakan anagram; yakni memiliki sebaran frekuensi karakter pembentuk yang persis sama, meskipun posisi hurufnya dikombinasikan ulang.

Python 
def deteksi_anagram(kata1, kata2):
    if len(kata1) != len(kata2): return False

    kamus_huruf = {}
    for char in kata1:
        if char in kamus_huruf: kamus_huruf[char] += 1
        else: kamus_huruf[char] = 1

    for char in kata2:
        if char in kamus_huruf: kamus_huruf[char] -= 1
        else: return False

    for count in kamus_huruf.values():
        if count != 0: return False
    return True
07 Linked List (Senarai Berantai) ▾

Node Head Traversal Linked List adalah struktur data dinamis yang disusun oleh sekumpulan objek bernama Node yang saling terhubung. Berbeda dengan tipe data List bawaan yang ukurannya diatur otomatis, struktur ini mengalokasikan ukuran secara dinamis dari awal tanpa penetapan ukuran tetap. Setiap node menyimpan dua informasi utama: data (nilai) dan referensi (pointer) yang menunjuk ke lokasi data berikutnya.

Operasi Dasar Linked List

Python 
# Implementasi Linked List Prosedural (Memanfaatkan Dictionary sebagai Node)
def buat_node(data_awal):
    return {'data': data_awal, 'next': None}

def buat_linked_list():
    return {'head': None}

def is_empty(ll):
    return ll['head'] is None

def add(ll, item):
    temp = buat_node(item)
    temp['next'] = ll['head']
    ll['head'] = temp

def size(ll):
    curent = ll['head']
    count = 0
    while curent is not None:
        count += 1
        curent = curent['next']
    return count

def search(ll, item):
    curent = ll['head']
    found = False
    while curent is not None and not found:
        if curent['data'] == item:
            found = True
        else:
            curent = curent['next']
    return found
08 Binary Tree & Parse Tree ▾

Root Leaf Parse Tree Binary Tree adalah struktur data berbentuk non-linear. Konsep dasarnya terbalik dari ekosistem pohon biologi, di mana root (akar) berada di tingkat paling atas dan memiliki edge (cabang) yang menjalar ke bawah sampai mencapai leaf (daun). Dalam versi biner, setiap parent dibatasi maksimal hanya memiliki dua anak: anak kiri (left child) dan anak kanan (right child).

1. Pembentukan Binary Tree & Percabangan

Python 
# Implementasi Pohon Biner Prosedural
def buat_binary_tree(root_val):
    return {'key': root_val, 'left_child': None, 'right_child': None}

def insert_left(tree, new_node):
    if tree['left_child'] is None:
        tree['left_child'] = buat_binary_tree(new_node)
    else:
        t = buat_binary_tree(new_node)
        t['left_child'] = tree['left_child']
        tree['left_child'] = t

def insert_right(tree, new_node):
    if tree['right_child'] is None:
        tree['right_child'] = buat_binary_tree(new_node)
    else:
        t = buat_binary_tree(new_node)
        t['right_child'] = tree['right_child']
        tree['right_child'] = t

def get_left_child(tree): return tree['left_child']
def get_right_child(tree): return tree['right_child']
def set_root_val(tree, obj): tree['key'] = obj
def get_root_val(tree): return tree['key']

2. Aplikasi Parse Tree

Digunakan untuk memecahkan dan mengevaluasi persamaan matematika. Mengkombinasikan struktur penyimpanan Stack dan memori hirarkis dari Binary Tree untuk menyimpan alur turun-naik (prioritas kurung).

Python 
# Algoritma Pembangunan Parse Tree (menggabungkan stack & tree)
def build_parse_tree(math_exp):
    token_list = math_exp.split()
    parent_stack = buat_stack() 
    exp_tree = buat_binary_tree('')
    push(parent_stack, exp_tree)
    current_tree = exp_tree
    
    for i in token_list:
        if i == '(':
            insert_left(current_tree, '')
            push(parent_stack, current_tree)
            current_tree = get_left_child(current_tree)
        elif i not in ['+', '-', '*', '/', ')']:
            set_root_val(current_tree, int(i))
            current_tree = pop(parent_stack)
        elif i in ['+', '-', '*', '/']:
            set_root_val(current_tree, i)
            insert_right(current_tree, '')
            push(parent_stack, current_tree)
            current_tree = get_right_child(current_tree)
        elif i == ')':
            current_tree = pop(parent_stack)
            
    return exp_tree
