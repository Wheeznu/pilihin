1. Stack (Tumpukan) & Simulasi

Konsep: Sebuah kumpulan item yang terurut di mana penambahan item baru dan penghapusan item yang sudah ada selalu dilakukan pada ujung yang sama (top). Prinsip pengurutan ini disebut sebagai LIFO (Last In, First Out). 

class Stack:
    def __init__(self): self.items = []
    def isEmpty(self): return self.items == []
    def push(self, item): self.items.append(item)
    def pop(self): return self.items.pop()
    def peek(self): return self.items[-1]
    def size(self): return len(self.items)

Studi Kasus 1: Pengecekan Palindrome 

kata = "KATAK"

2. Queue & Deque (Antrean)

Queue (FIFO): Penambahan item di belakang (rear) dan penghapusan di depan (front). 

class Queue:
    def __init__(self): self.items = []
    def enqueue(self, item): self.items.insert(0, item)
    def dequeue(self): return self.items.pop()

Deque (Double-Ended Queue): Item dapat ditambahkan atau dihapus dari kedua ujung. 

class DeQueue:
    def __init__(self): self.items = []
    def addRear(self, item): self.items.insert(0, item)
    def addFront(self, item): self.items.append(item)
    def removeFront(self): return self.items.pop()
    def removeRear(self): return self.items.pop(0)
3. Evaluasi Ekspresi Postfix

Pemanfaatan Stack untuk mengonversi dan menghitung hasil dari notasi postfix. 

def evaluate_postfix(postfix_expr):
    stack = []
    for char in postfix_expr:
        if char.isdigit(): 
            stack.append(int(char))
        else:
            val1 = stack.pop()
            val2 = stack.pop()
            if char == '+': stack.append(val2 + val1)
            elif char == '-': stack.append(val2 - val1)
            elif char == '*': stack.append(val2 * val1)
            elif char == '/': stack.append(val2 / val1)
    return stack.pop()
4. Algoritma Sorting (Pengurutan)

Bubble Sort: Menukar elemen yang bersebelahan. 

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

Selection Sort: Mencari elemen terkecil dan memindahkannya ke kiri. 

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]: min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]

Insertion Sort: Menyisipkan kartu ke posisi yang tepat. 

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

Quick Sort: Menggunakan Pivot (Divide and Conquer). 

def quick_sort(arr):
    if len(arr) <= 1: return arr
    else:
        pivot = arr[len(arr) // 2]
        left = [x for x in arr if x < pivot]
        middle = [x for x in arr if x == pivot]
        right = [x for x in arr if x > pivot]
        return quick_sort(left) + middle + quick_sort(right)

5. Algoritma Searching (Pencarian)

Sequential / Linear Search:

def sequentialSearch(alist, item):
    pos = 0; found = False
    while pos < len(alist) and not found:
        if alist[pos] == item: found = True
        else: pos = pos + 1
    return found

Binary Search:

def binarySearch(alist, item):
    first, last = 0, len(alist) - 1
    found = False
    while first <= last and not found:
        midpoint = (first + last) // 2
        if alist[midpoint] == item: found = True
        elif item < alist[midpoint]: last = midpoint - 1
        else: first = midpoint + 1
    return found
6. Implementasi Hashing Prosedural

Konsep Hash Table: Struktur data tempat penyimpanan elemen berdasarkan nilai kunci pencarian (hash key) guna mempercepat proses pencarian data (mencapai O(1)).

⚠️ Aturan Batasan Pengodean Tugas:

  • Dilarang keras menggunakan arsitektur OOP / class. Wajib menggunakan fungsi prosedural.
  • Sistem input tidak digabungkan; ditangani per individu prompt (satu persatu).
  • Boleh menggunakan parameter num dalam operasi perhitungan nilai/biner.

1. Linear Probing

table = [None] * 11        
value = 44                 
collision = value % 11     

table[collision] = 999

if ind >= len(table): ind = 0

while (ind <= len(table) - 1) and not found:
    if table[ind] == None:
        found = True; table[ind] = value
    ind = ind + 1

2. Quadratic Probing
